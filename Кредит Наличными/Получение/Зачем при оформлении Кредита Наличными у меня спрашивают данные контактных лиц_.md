# Зачем при оформлении Кредита Наличными у меня спрашивают данные контактных лиц?

- Дата создания: 18.08.2016
- Дата обновления: 20.02.2019
- Полезность: 92%

## Ответ

Контакты необходимы для подтверждения данных, которые вы указываете в анкете при получении кредита. Обязательства по кредиту контактные лица не несут.
