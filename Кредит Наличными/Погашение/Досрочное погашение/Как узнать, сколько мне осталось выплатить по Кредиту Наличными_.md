# Как узнать, сколько мне осталось выплатить по Кредиту Наличными?

- Дата создания: 18.08.2016
- Дата обновления: 14.08.2018
- Полезность: 91%

## Ответ

Вы можете посмотреть сумму для полного досрочного погашения не выходя из дома — на Kaspi.kz в сервисе «Мой Банк» во вкладке «Кредит». Также это можно сделать через любой из Kaspi Терминалов или через Kaspi Банкоматы.
