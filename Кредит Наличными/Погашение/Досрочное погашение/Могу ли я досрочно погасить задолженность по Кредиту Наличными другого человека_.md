# Могу ли я досрочно погасить задолженность по Кредиту Наличными другого человека?

- Дата создания: 18.08.2016
- Дата обновления: 22.11.2019
- Полезность: 93%

## Ответ

Да. Досрочно погасить кредит возможно без комиссий, штрафов и очередей через любой Kaspi Терминал или Kaspi Банкомат. Для этого нужно знать ИИН клиента, на которого оформлен кредит.
