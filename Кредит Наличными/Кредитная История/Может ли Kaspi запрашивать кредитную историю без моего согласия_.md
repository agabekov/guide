# Может ли Kaspi запрашивать кредитную историю без моего согласия?

- Дата создания: 18.08.2016
- Дата обновления: 15.12.2020
- Полезность: 92%

## Ответ

Нет. Проверка кредитной истории осуществляется только с вашего письменного согласия. Если согласие было предоставлено ранее, при повторном оформлении кредита новое согласие не требуется.
