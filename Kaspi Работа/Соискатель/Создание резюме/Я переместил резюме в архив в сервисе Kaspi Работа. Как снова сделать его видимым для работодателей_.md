# Я переместил резюме в архив в сервисе Kaspi Работа. Как снова сделать его видимым для работодателей?

- Дата создания: 04.03.2025
- Дата обновления: 05.03.2025
- Полезность: 96%

## Ответ

В сервисе Kaspi Работа перейдите во вкладку Кабинет → Архив. Выберите резюме из списка и нажмите «Восстановить». Оно станет активным и видимым для работодателей.
