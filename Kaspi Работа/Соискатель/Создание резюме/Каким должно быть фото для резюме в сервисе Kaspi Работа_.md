# Каким должно быть фото для резюме в сервисе Kaspi Работа?

- Дата создания: 04.03.2025
- Дата обновления: 05.03.2025
- Полезность: 88%

## Ответ

Соблюдайте следующие требования: светлый, однородный фон; равномерное освещение; прямой взгляд в камеру; нейтральное выражение лица; отсутствие головного убора.
