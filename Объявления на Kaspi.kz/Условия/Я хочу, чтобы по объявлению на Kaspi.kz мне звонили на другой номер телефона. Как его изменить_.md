# Я хочу, чтобы по объявлению на Kaspi.kz мне звонили на другой номер телефона. Как его изменить?

- Дата создания: 21.10.2022
- Дата обновления: 26.03.2024
- Полезность: 92%

## Ответ

Вы можете изменить номер телефона после подачи объявления: перейдите в раздел «Кабинет», выберите нужное, нажмите «Редактировать», введите нужный номер и подтвердите действие с помощью кода из SMS. При размещении нового объявления для покупателей автоматически будет указан тот, с которым вы зарегистрировались в <ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>. Вы можете оставить его или ввести другой номер.
