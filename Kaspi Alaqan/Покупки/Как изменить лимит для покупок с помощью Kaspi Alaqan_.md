# Как изменить лимит для покупок с помощью Kaspi Alaqan?

- Дата создания: 21.11.2025
- Дата обновления: 26.11.2025
- Полезность: 100%

## Ответ

Вы можете изменить лимит в приложении Kaspi.kz: Настройки → Kaspi Alaqan → Лимит на оплату в месяц. Новый лимит можно использовать сразу после изменения.
