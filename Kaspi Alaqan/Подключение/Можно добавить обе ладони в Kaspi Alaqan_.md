# Можно добавить обе ладони в Kaspi Alaqan?

- Дата создания: 29.10.2025
- Дата обновления: 02.11.2025
- Полезность: 100%

## Ответ

Нет. В Kaspi Alaqan можно использовать только одну ладонь. Если вы хотите добавить другую, можно поменять руку. Для этого поднесите ладонь к камере на Kaspi Alaqan и следуйте дальнейшим шагам на экране.
