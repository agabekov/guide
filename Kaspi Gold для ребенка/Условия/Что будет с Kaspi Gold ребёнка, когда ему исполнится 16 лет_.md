# Что будет с Kaspi Gold ребёнка, когда ему исполнится 16 лет?

- Дата создания: 03.12.2020
- Дата обновления: 23.05.2023
- Полезность: 93%

## Ответ

Kaspi Gold для ребёнка будет активна до истечения срока действия, который указан на лицевой стороне карты. С 16 лет подросток может открыть Kaspi Gold самостоятельно. Для этого родителю или опекуну нужно закрыть детскую карту в мобильном приложении Kaspi.kz. Затем, чтобы открыть новую карту, подростку нужно в мобильном приложении Kaspi.kz выбрать «Открыть Kaspi Gold» на главной странице и следовать простым дальнейшим шагам. Получить свою именную Kaspi Gold подросток может в любом Kaspi Картомате. Для этого понадобится только мобильный телефон.
