# Ребенок может самостоятельно делать покупки в Магазине на Kaspi.kz?

- Дата создания: 31.05.2024
- Дата обновления: 01.06.2024
- Полезность: 89%

## Ответ

Нет. Оформить заказ в Магазине на Kaspi.kz может только взрослый. Ребенок может выбирать товары; добавлять их в «Избранное»; отправлять ссылки на них с помощью любого мессенджера и электронной почты. Для этого нужно нажать «Поделиться товаром» в карточке товара.
