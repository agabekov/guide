# Как зарегистрироваться в мобильном приложении Kaspi.kz с Kaspi Gold для ребёнка?

- Дата создания: 28.04.2023
- Дата обновления: 03.05.2023
- Полезность: 94%

## Ответ

Скачайте и установите мобильное приложение Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'> на смартфон ребёнка. Укажите его номер мобильного и введите код из SMS. Затем для подтверждения регистрации отсканируйте Kaspi QR на экране смартфона ребёнка с помощью своего приложения Kaspi.kz или введите номер Kaspi Gold для ребёнка. Обратите внимание: подтвердить регистрацию по Kaspi QR может только родитель, который открыл Kaspi Gold для ребёнка.
