# Я оплатил картой несколько товаров в одном магазине. Я могу вернуть только один из товаров?

- Дата создания: 23.12.2020
- Дата обновления: 23.05.2023
- Полезность: 77%

## Ответ

Чтобы вернуть один из товаров, попроси родителя или опекуна обратиться к продавцу с чеком. Если тебе уже исполнилось 14 лет, можешь самостоятельно вернуть покупку.
