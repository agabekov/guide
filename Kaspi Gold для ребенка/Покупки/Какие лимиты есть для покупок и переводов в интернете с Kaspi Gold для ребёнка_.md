# Какие лимиты есть для покупок и переводов в интернете с Kaspi Gold для ребёнка?

- Дата создания: 14.10.2024
- Дата обновления: 15.10.2024
- Полезность: 95%

## Ответ

Вы можете установить для покупок в интернете с Kaspi Gold для ребенка ограничение на месяц: 5 000 ₸ 10 000 ₸ 15 000 ₸ 25 000 ₸ 50 000 ₸ Также можно открыть доступ без лимита. Для этого в своем мобильном приложении Kaspi.kz перейдите в Мой Банк → Kaspi Gold ребенка → Действия → Оплата в интернете. Выбранный лимит действует постоянно до того момента, пока вы сами его не отключите или не измените. Исключение — лимит без ограничений. Его нужно активировать заново в начале каждого месяца.
