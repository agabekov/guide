# Зачем нужны лимиты для покупок и переводов в интернете с Kaspi Gold для ребёнка?

- Дата создания: 26.10.2022
- Дата обновления: 26.10.2022
- Полезность: 97%

## Ответ

Лимит необходим, чтобы ваш ребёнок мог безопасно совершать покупки и переводы в интернете со своей Kaspi Gold для ребёнка. Выбранный лимит будет автоматически продлеваться каждый месяц. Исключение — лимит без ограничений. Его нужно активировать заново в начале каждого месяца.
