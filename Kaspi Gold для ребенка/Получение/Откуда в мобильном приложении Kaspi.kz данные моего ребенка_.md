# Откуда в мобильном приложении Kaspi.kz данные моего ребенка?

- Дата создания: 22.02.2022
- Дата обновления: 23.05.2023
- Полезность: 95%

## Ответ

Когда открываете Kaspi Gold для ребенка, вы подписываете документ «Согласие на сбор и обработку персональных данных» с помощью электронной подписи Kaspi. Это нужно для того, чтобы ФИО и ИИН отображались в мобильном приложении Kaspi.kz и вам не пришлось вводить их вручную. Информация о ребенке поступает в мобильное приложение Kaspi.kz из государственной базы данных с вашего разрешения.
