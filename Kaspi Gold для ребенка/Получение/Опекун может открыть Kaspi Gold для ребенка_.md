# Опекун может открыть Kaspi Gold для ребенка?

- Дата создания: 22.05.2023
- Дата обновления: 23.05.2023
- Полезность: 95%

## Ответ

Да. Опекун может открыть Kaspi Gold для ребенка в мобильном приложении Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>. Для этого на главной странице нажмите «Kaspi Gold для ребенка» → Оформить Kaspi Gold для ребенка и следуйте простым дальнейшим шагам.
