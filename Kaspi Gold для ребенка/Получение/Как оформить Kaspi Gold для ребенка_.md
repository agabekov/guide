# Как оформить Kaspi Gold для ребенка?

- Дата создания: 03.12.2020
- Дата обновления: 14.08.2024
- Полезность: 95%

## Ответ

Оформить Kaspi Gold для ребенка вы можете в своем мобильном приложении Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>. Для этого на главной странице нажмите «Kaspi Gold для ребенка» → Оформить Kaspi Gold для ребенка. Подпишите документ «Согласие на сбор и обработку персональных данных» с помощью ЭЦП Kaspi. Это нужно для того, чтобы данные вашего ребенка отобразились в мобильном приложении Kaspi.kz. Если у вас несколько детей, выберите того, кому вы оформляете Kaspi Gold, заполните поле с номером телефона ребенка и подтвердите информацию. Ребенку придет SMS с кодом. Введите его и следуйте дальнейшим шагам на экране. После оформления в мобильном приложении Kaspi.kz заберите карту в Kaspi Картомате. Понадобится только ваш мобильный телефон. Ребенка брать с собой не обязательно. Выпуск именной Kaspi Gold для вашего ребенка займет всего 60 секунд.
