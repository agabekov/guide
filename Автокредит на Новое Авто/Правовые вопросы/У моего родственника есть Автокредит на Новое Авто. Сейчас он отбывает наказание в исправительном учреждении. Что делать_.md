# У моего родственника есть Автокредит на Новое Авто. Сейчас он отбывает наказание в исправительном учреждении. Что делать?

- Дата создания: 18.08.2016
- Дата обновления: 05.07.2022
- Полезность: 89%

## Ответ

Заключение в исправительном учреждении (тюрьме) не является основанием для освобождения от обязательств по кредиту. Платежи можно совершить через любой из Kaspi Терминалов по всей стране или в Kaspi Банкоматах. Для этого нужен только номер ИИН.
