# Могу ли я досрочно погасить задолженность по Автокредиту на Новое Авто другого человека?

- Дата создания: 18.08.2016
- Дата обновления: 05.07.2022
- Полезность: 100%

## Ответ

Да. Досрочно погасить кредит можно через любой Kaspi Терминал или через Kaspi Банкомат. Для этого нужно знать ИИН клиента, на которого оформлен кредит.
