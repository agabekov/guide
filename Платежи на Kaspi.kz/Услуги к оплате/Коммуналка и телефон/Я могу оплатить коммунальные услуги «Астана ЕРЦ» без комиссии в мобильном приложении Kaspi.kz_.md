# Я могу оплатить коммунальные услуги «Астана ЕРЦ» без комиссии в мобильном приложении Kaspi.kz?

- Дата создания: 18.08.2016
- Дата обновления: 19.07.2022
- Полезность: 96%

## Ответ

Да. В мобильном приложении Kaspi.kz можно оплатить коммунальные услуги «Астана ЕРЦ» без комиссии. Для этого выберите сервис «Платежи» → Коммуналка и телефон → Квитанции за квартиру → Астана ЕРЦ и введите номер абонента, который указан на бумажной квитанции. В открывшемся окне выберите необходимые услуги для оплаты и подтвердите платёж. После этого вы получите квитанцию. Во вкладке «История» её можно отправить с помощью любой программы, установленной на вашем смартфоне. В следующем месяце квитанция отобразится в разделе «К оплате», и вводить номер абонента не понадобится.
