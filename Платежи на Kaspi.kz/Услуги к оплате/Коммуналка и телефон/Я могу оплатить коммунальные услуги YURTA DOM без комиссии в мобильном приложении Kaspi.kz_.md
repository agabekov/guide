# Я могу оплатить коммунальные услуги YURTA DOM без комиссии в мобильном приложении Kaspi.kz?

- Дата создания: 18.08.2016
- Дата обновления: 19.02.2024
- Полезность: 97%

## Ответ

Да. В приложении Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'> жители города Алматы могут оплатить коммунальные услуги YURTA DOM без комиссии. Для этого выберите сервис «Платежи» → Коммуналка и телефон → Квитанции за квартиру → YURTA DOM и введите номер лицевого счета, который указан на бумажной квитанции. В открывшемся окне выберите необходимые услуги для оплаты. Затем подтвердите платёж. После этого вы получите квитанцию об оплате и можете перейти во вкладку «История». В мобильном приложении Kaspi.kz квитанцию можно отправить с помощью любой программы, установленной на вашем смартфоне. В следующий раз оплатить эту услугу можно во вкладке «Мои платежи» или «История», тогда вводить номер лицевого счета не понадобится.
