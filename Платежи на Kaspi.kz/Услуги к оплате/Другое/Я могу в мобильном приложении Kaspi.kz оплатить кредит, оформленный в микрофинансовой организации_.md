# Я могу в мобильном приложении Kaspi.kz оплатить кредит, оформленный в микрофинансовой организации?

- Дата создания: 20.08.2019
- Дата обновления: 24.05.2024
- Полезность: 96%

## Ответ

Да. В <ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'> вы можете совершить платеж в пользу микрофинансовых организаций (МФО) без комиссии. Для этого откройте сервис Платежи → Финансовые услуги → МФО. Выберите нужную организацию, введите ИИН и следуйте дальнейшим шагам на экране. В месяц можно внести максимум 3 миллиона тенге по каждой МФО.
