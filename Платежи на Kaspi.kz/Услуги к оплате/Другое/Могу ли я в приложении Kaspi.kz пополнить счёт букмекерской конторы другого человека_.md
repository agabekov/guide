# Могу ли я в приложении Kaspi.kz пополнить счёт букмекерской конторы другого человека?

- Дата создания: 12.08.2022
- Дата обновления: 15.08.2022
- Полезность: 88%

## Ответ

Да. Для этого откройте сервис «Платежи» → Другое → Букмекерские конторы. Выберите нужную, введите логин или номер счёта, который хотите пополнить, и нажмите продолжить.
