# Как в приложении Kaspi.kz узнать, за что и когда выписан штраф ПДД?

- Дата создания: 18.08.2016
- Дата обновления: 03.06.2025
- Полезность: 90%

## Ответ

В приложении Kaspi.kz в сервисе Платежи → Штрафы и налоги → Штрафы вы можете проверить факт нарушения ПДД: по ИИН; по госномеру авто; по БИН; по номеру паспорта для лиц без гражданства РК. Если штрафы есть, у вас откроется список с подробной информацией о каждом из них: дата, детали нарушения, сумма к оплате. Посмотреть протокол можно только по ИИН: при оформлении сотрудником дорожно-патрульной полиции — по данным водителя, при фиксации камерой — владельца машины.
