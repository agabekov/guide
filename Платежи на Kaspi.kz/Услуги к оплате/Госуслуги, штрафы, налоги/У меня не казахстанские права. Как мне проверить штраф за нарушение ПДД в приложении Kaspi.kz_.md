# У меня не казахстанские права. Как мне проверить штраф за нарушение ПДД в приложении Kaspi.kz?

- Дата создания: 18.08.2016
- Дата обновления: 02.06.2025
- Полезность: 88%

## Ответ

Иностранные граждане, у которых есть Kaspi Gold или Счет Kaspi Pay, могут проверить и оплатить штраф по номеру и дате выдачи загранпаспорта. Для этого перейдите в сервис Платежи → Штрафы и налоги → Штрафы → Штрафы по номеру паспорта для лиц без гражданства РК. Если штрафы есть, откроется список с подробной информацией о каждом из них: дата, суть нарушения, сумма к оплате.
