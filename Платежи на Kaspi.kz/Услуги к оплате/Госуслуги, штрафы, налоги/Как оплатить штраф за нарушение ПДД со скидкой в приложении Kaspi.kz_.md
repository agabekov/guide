# Как оплатить штраф за нарушение ПДД со скидкой в приложении Kaspi.kz?

- Дата создания: 18.08.2016
- Дата обновления: 02.06.2025
- Полезность: 94%

## Ответ

Штраф за нарушение ПДД можно оплатить со скидкой 50% в течение установленного законом срока. Для этого в приложении Kaspi.kz перейдите в сервис Платежи → Штрафы и налоги → Штрафы. Вы можете оплатить нарушение: по ИИН; по госномеру авто; по БИН; по номеру паспорта для лиц без гражданства РК. Если штраф выписывает сотрудник дорожно-патрульной полиции, к протоколу прикладывается скидочный талон, в котором указана сумма штрафа и срок действия скидки. По нарушениям, зафиксированным камерами, владельцу машины придет сообщение в приложении Kaspi.kz. Скидка действует 7 дней с даты получения уведомления.
