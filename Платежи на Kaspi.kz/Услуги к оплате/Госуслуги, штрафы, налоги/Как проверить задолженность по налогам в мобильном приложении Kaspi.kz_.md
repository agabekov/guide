# Как проверить задолженность по налогам в мобильном приложении Kaspi.kz?

- Дата создания: 05.09.2018
- Дата обновления: 31.07.2025
- Полезность: 89%

## Ответ

В мобильном приложении Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'> перейдите в сервис «Госуслуги» → «Налоги» и выберите услугу «Долги по налогам». • Введите ИИН и нажмите «Проверить». Если у вас есть задолженности, откроется список с информацией о каждой из них: вид налога, сумма долга, пеня. • Подтвердите платёж. Сумма поступит на следующий рабочий день, после чего задолженность будет погашена. После этого вы получите квитанцию. Во вкладке «История» её можно отправить с помощью любой программы, установленной на вашем смартфоне. В следующий раз, если возникнет долг, счёт на оплату налога отобразится в разделе «К оплате».
