# Как оплатить пеню по налогу на транспорт на Kaspi.kz?

- Дата создания: 18.08.2016
- Дата обновления: 19.09.2019
- Полезность: 98%

## Ответ

В разделе Платежи → Штрафы и платежи в бюджет → НалогиНалогиНалоги выберите услугу «Налог для физ. лиц по реквизитам»«Налог для физ. лиц по реквизитам»«Налог для физ. лиц по реквизитам». Проверьте, что город указан верно, и выберите тип платежа «Пеня по налогу на транспорт с физических лиц» в выпадающем меню. Выберите Управление государственных доходов и район, введите ФИО и ИИН налогоплательщика. Укажите сумму, выберите способ оплаты и подтвердите платёж. После этого вы получите квитанцию об оплате и можете перейти во вкладку «История». Здесь в мобильном приложении Kaspi.kz квитанцию можно отправить с помощью любой программы, установленной на вашем смартфоне.
