# На меня оформлены несколько квартир. Я могу оплатить долг по налогу на имущество только за одну из них в мобильном приложении Kaspi.kz?

- Дата создания: 05.09.2018
- Дата обновления: 01.07.2025
- Полезность: 94%

## Ответ

Нет. В мобильном приложении Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'> отображается общий долг по определённому типу налога. Если у вас две квартиры, то сумма налога будет на обе сразу. Чтобы оплатить начисления по одной из квартир, нужно обратиться в управление государственных доходов по месту нахождения имущества.
