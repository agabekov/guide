# Как оплатить административный штраф в мобильном приложении Kaspi.kz?

- Дата создания: 18.08.2016
- Дата обновления: 15.12.2021
- Полезность: 82%

## Ответ

В мобильном приложении Kaspi.kz перейдите в сервис «Платежи» → Госуслуги, штрафы, налоги → Штрафы. Оплатить штраф можно тремя способами: по ИИН; по БИН; по госномеру авто. Если штрафы есть, то откроется список с подробной информацией о каждом из них: дата, причина, сумма штрафа. Затем выберите штраф и подтвердите платёж. После этого вы получите квитанцию об оплате и можете перейти во вкладку «История». Здесь в мобильном приложении Kaspi.kz квитанцию можно отправить с помощью любой программы, установленной на вашем смартфоне.
