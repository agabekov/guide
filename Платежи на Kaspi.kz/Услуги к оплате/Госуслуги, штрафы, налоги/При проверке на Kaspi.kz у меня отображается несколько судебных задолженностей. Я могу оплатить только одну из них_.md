# При проверке на Kaspi.kz у меня отображается несколько судебных задолженностей. Я могу оплатить только одну из них?

- Дата создания: 18.08.2016
- Дата обновления: 21.12.2020
- Полезность: 90%

## Ответ

Да. Вы можете выбрать для оплаты только одну из задолженностей. Для этого снимите галочки с тех исполнительных производств, которые вы не будете оплачивать сейчас.
