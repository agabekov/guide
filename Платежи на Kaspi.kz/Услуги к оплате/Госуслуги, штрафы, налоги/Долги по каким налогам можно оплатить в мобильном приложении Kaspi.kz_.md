# Долги по каким налогам можно оплатить в мобильном приложении Kaspi.kz?

- Дата создания: 05.09.2018
- Дата обновления: 19.07.2022
- Полезность: 92%

## Ответ

В мобильном приложении Kaspi.kz можно проверить и оплатить долг по всем налогам для физических лиц: Платежи → Госуслуги, штрафы, налоги → НалогиНалогиНалоги → Долги по налогамДолги по налогамДолги по налогам. Исключения: социальные отчисления, обязательные пенсионные взносы и социальное медицинское страхование. Чтобы узнать задолженность по этим платежам, нужно обратиться в ГЦВП. Долги по налогам для юридических лиц проверить на Kaspi.kz нельзя, но оплатить можно. Для этого перейдите в Платежи → Госуслуги, штрафы, налоги → Налоги → Налог для юр. лиц по реквизитам и заполните необходимые поля. В следующий раз, если возникнет долг, счёт на оплату налога отобразится в разделе «К оплате».
