# Как оплатить пеню по земельному налогу на Kaspi.kz?

- Дата создания: 18.08.2016
- Дата обновления: 19.09.2019
- Полезность: 90%

## Ответ

В мобильном приложении Kaspi.kz вы можете без комиссии оплатить два вида пени по земельному налогу : пеню по земельному налогу на земли населенных пунктов пеню по земельному налогу на земли ненаселенных пунктов Для оплаты перейдите в раздел Платежи → Штрафы и платежи в бюджет → НалогиНалогиНалоги и выберите услугу «Налог для физ. лиц по реквизитам»«Налог для физ. лиц по реквизитам»«Налог для физ. лиц по реквизитам». Проверьте, что город указан верно, и выберите нужный тип налога в выпадающем меню. Выберите Управление государственных доходов и район, введите ФИО и ИИН налогоплательщика. Укажите сумму, выберите способ оплаты и подтвердите платёж. После этого вы получите квитанцию об оплате и можете перейти во вкладку «История». Здесь в мобильном приложении Kaspi.kz квитанцию можно отправить с помощью любой программы, установленной на вашем смартфоне.
