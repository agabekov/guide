# При проверке на Kaspi.kz у меня отображается несколько долгов по налогам. Я могу оплатить только один из них?

- Дата создания: 05.09.2018
- Дата обновления: 25.01.2019
- Полезность: 94%

## Ответ

Да. Вы можете выбрать для оплаты только один из долгов. Для этого снимите галочки с тех задолженностей, которые вы не будете оплачивать сейчас.
