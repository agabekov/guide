# Как посмотреть протокол штрафа за нарушение ПДД в приложении Kaspi.kz?

- Дата создания: 02.06.2025
- Дата обновления: 03.06.2025
- Полезность: 80%

## Ответ

Есть два способа: Если штраф оплачен Платежи → История, выберите нужную операцию и нажмите «Посмотреть протокол»; Если штраф не оплачен Платежи → Штрафы и налоги → Штрафы, откройте подробную информацию о штрафе и нажмите «Посмотреть протокол». Посмотреть протокол можно только по ИИН: при оформлении сотрудником дорожно-патрульной полиции, - по данным водителя, при фиксации камерой — владельца машины.
