# Какие ограничения при оплате мобильной связи в приложении Kaspi.kz?

- Дата создания: 18.08.2016
- Дата обновления: 27.12.2024
- Полезность: 77%

## Ответ

Максимальные суммы в тенге для каждого из операторов: в месяц — 50 000; за один раз — 20 000. Лимит обнуляется каждый месяц первого числа. Оплатить мобильную связь без ограничений по сумме можно через Kaspi Терминалы. За эту услугу взимается комиссия.
