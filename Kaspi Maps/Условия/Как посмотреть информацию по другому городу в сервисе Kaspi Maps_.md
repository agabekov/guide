# Как посмотреть информацию по другому городу в сервисе Kaspi Maps?

- Дата создания: 18.08.2016
- Дата обновления: 24.01.2025
- Полезность: 86%

## Ответ

В мобильном приложении Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'> откройте сервис Kaspi Maps и нажмите иконку геолокации. Выберите нужный город из появившегося списка. Теперь вы можете посмотреть информацию по другому городу в сервисе Kaspi Maps.
