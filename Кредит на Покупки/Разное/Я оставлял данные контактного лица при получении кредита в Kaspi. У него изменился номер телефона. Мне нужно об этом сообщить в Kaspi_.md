# Я оставлял данные контактного лица при получении кредита в Kaspi. У него изменился номер телефона. Мне нужно об этом сообщить в Kaspi?

- Дата создания: 18.08.2016
- Дата обновления: 27.08.2020
- Полезность: 59%

## Ответ

Нет. Данные контактных лиц нужны для подтверждения информации, которую вы указываете в анкете при получении кредита. Обязательств по кредиту контактные лица не несут. Поэтому сообщать Kaspi об изменении их контактных данных не нужно.
