# Почему рассрочку можно оформить только на 3 месяца, раньше больше можно было?

- Дата создания: 18.08.2016
- Дата обновления: 15.07.2022
- Полезность: 80%

## Ответ

Новые условия Кредита на Покупки в Kaspi выгодны для клиента, так как: условия единые во всех магазинах рассрочка на 3 месяца всегда, а не по акции первоначальный взнос не требуется нет страховки
