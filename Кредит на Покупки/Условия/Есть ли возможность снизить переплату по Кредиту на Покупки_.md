# Есть ли возможность снизить переплату по Кредиту на Покупки?

- Дата создания: 18.08.2016
- Дата обновления: 13.08.2025
- Полезность: 88%

## Ответ

У вас есть возможность оформить Кредит на Покупки без процентов на 3 месяца. Для снижения переплаты вы можете в любое время частично или полностью досрочно погасить кредит без комиссий и штрафов. Минимальная сумма частичного досрочного погашения равна сумме одного ежемесячного платежа. При досрочном погашении вознаграждение по кредиту пересчитывается на реальное количество дней пользования кредитом.
