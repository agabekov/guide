# Я уменьшил лимит Kaspi Red+. Почему доступная для покупок сумма стала меньше?

- Дата создания: 22.09.2022
- Дата обновления: 21.10.2024
- Полезность: 88%

## Ответ

Когда вы уменьшаете лимит, доступная для покупок сумма тоже становится меньше. Если у вас есть задолженность по Kaspi Red+, сумма увеличится после оплаты ежемесячного счета.
