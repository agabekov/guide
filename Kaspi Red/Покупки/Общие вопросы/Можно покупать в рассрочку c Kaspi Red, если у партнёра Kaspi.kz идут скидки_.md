# Можно покупать в рассрочку c Kaspi Red, если у партнёра Kaspi.kz идут скидки?

- Дата создания: 18.08.2016
- Дата обновления: 11.11.2022
- Полезность: 94%

## Ответ

Да. Вы можете оплачивать в рассрочку 0% с Kaspi Red любые товары или услуги у партнёров Kaspi.kz в рамках доступной суммы для покупок.
