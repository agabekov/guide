# Я могу с Kaspi Red оплатить покупку в магазине, который не является партнёром Kaspi.kz?

- Дата создания: 18.08.2016
- Дата обновления: 22.01.2025
- Полезность: 82%

## Ответ

Нет. Покупки с Kaspi Red можно совершать только у партнёров Kaspi.kz и в Kaspi Магазине. Посмотреть информацию о партнёрах можно в сервисе Мой Банк → Kaspi Red+ → Где купить? → Все партнеры Kaspi Red+.
