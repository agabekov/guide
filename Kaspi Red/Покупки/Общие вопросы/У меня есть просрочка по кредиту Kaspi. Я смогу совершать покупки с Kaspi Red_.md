# У меня есть просрочка по кредиту Kaspi. Я смогу совершать покупки с Kaspi Red?

- Дата создания: 18.08.2016
- Дата обновления: 18.08.2016
- Полезность: 86%

## Ответ

Нет. Вам необходимо погасить просроченную задолженность по кредиту Kaspi. После погашения Kaspi Red будет доступна для совершения покупок.
