# Как узнать сумму следующего ежемесячного счёта Kaspi Red?

- Дата создания: 11.04.2022
- Дата обновления: 03.06.2024
- Полезность: 93%

## Ответ

В <ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'> перейдите в сервис Мой Банк → Kaspi Red → Инфо → Календарь счетов. Нажмите на нужный месяц, чтобы увидеть детали счёта.
