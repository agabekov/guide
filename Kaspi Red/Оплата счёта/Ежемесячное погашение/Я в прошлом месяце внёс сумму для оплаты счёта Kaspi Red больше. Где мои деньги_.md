# Я в прошлом месяце внёс сумму для оплаты счёта Kaspi Red больше. Где мои деньги?

- Дата создания: 18.08.2016
- Дата обновления: 03.06.2024
- Полезность: 56.00000000000001%

## Ответ

Есть два варианта: у вас остались неоплаченные покупки с Kaspi Red – остаток внесённой суммы пойдёт на оплату следующего счета. Доступная сумма увеличится только после полного погашения одной из покупок. покупки с Kaspi Red полностью оплачены – остаток средств сразу поступит на вашу Kaspi Gold. Обратите внимание: если вы не приняли новые условия Kaspi Red, доступный для покупок лимит восстановится сразу на внесенную сумму.
