# Где посмотреть счёт за покупки с Kaspi Red?

- Дата создания: 18.08.2016
- Дата обновления: 11.04.2022
- Полезность: 93%

## Ответ

Посмотреть подробную информацию о том, какие суммы и за какие покупки входят в счёт можно в <ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>: «Мой Банк» → Kaspi Red → Инфо → Календарь счетов. Нажмите на нужный месяц, чтобы увидеть детали счета.
