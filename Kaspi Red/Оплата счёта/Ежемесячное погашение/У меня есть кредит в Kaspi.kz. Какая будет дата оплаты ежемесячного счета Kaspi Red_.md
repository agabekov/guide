# У меня есть кредит в Kaspi.kz. Какая будет дата оплаты ежемесячного счета Kaspi Red?

- Дата создания: 18.08.2016
- Дата обновления: 03.06.2024
- Полезность: 91%

## Ответ

Датой оплаты счёта Kaspi Red будет дата ежемесячного платежа по кредитам. В день выставления счета Kaspi Red вам придёт сообщение в мобильном приложении Kaspi.kz с датой и суммой. Вы можете посмотреть подробную информацию о том, какие суммы и за какие покупки в него входят в <ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>: Мой Банк → Kaspi Red → Инфо → Календарь счетов. Нажмите на нужный месяц, чтобы увидеть детали. Оплатить его необходимо в плановую дату до 20:00. Сумма, внесенная позже, будет зачислена следующим днем.
