# При проверке на Kaspi.kz у меня отображается несколько административных штрафов. Я могу оплатить только один из них?

- Дата создания: 19.01.2021
- Дата обновления: 25.01.2019
- Полезность: 85%

## Ответ

Да. Вы можете выбрать для оплаты только один из штрафов. Для этого снимите галочки напротив тех штрафов, которые вы не будете оплачивать сейчас.
