# Я могу в мобильном приложении Kaspi.kz узнать, что мне выписали штраф за нарушение ПДД?

- Дата создания: 21.12.2020
- Дата обновления: 15.12.2021
- Полезность: 83%

## Ответ

Да. Для этого в мобильном приложении Kaspi.kz перейдите в сервис «Госуслуги» → Штрафы. Проверить наличие штрафа можно тремя способами: по ИИН; по БИН; по госномеру авто. После первой проверки эта услуга будет отображаться в разделе «Частые». Поэтому в следующий раз вы можете не вводить свои данные.
