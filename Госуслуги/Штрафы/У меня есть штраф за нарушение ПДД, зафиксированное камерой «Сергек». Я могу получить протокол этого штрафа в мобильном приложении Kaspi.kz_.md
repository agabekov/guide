# У меня есть штраф за нарушение ПДД, зафиксированное камерой «Сергек». Я могу получить протокол этого штрафа в мобильном приложении Kaspi.kz?

- Дата создания: 15.12.2021
- Дата обновления: 10.11.2022
- Полезность: 82%

## Ответ

Да. В мобильном приложении Kaspi.kz <ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>  в сервисе «Госуслуги» → Штрафы вы можете проверить факт нарушения ПДД и получить протокол штрафа: по ИИН; по БИН; по госномеру авто. Если штрафы есть, у вас откроется список с подробной информацией о каждом из них: датой, сутью нарушения и суммой к оплате. Обратите внимание: посмотреть протокол может только тот водитель, на чей ИИН пришёл штраф.
