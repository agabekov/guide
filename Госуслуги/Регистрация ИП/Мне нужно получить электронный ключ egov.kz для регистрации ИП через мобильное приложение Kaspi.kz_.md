# Мне нужно получить электронный ключ egov.kz для регистрации ИП через мобильное приложение Kaspi.kz?

- Дата создания: 21.12.2020
- Дата обновления: 21.12.2020
- Полезность: 94%

## Ответ

Нет. Для регистрации ИП через мобильное приложение Kaspi.kz достаточно подписать документы с помощью электронной подписи Kaspi, которую вы также можете получить в процессе регистрации ИП.
