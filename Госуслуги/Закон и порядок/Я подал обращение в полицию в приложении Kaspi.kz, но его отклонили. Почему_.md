# Я подал обращение в полицию в приложении Kaspi.kz, но его отклонили. Почему?

- Дата создания: 24.06.2025
- Дата обновления: 20.10.2025
- Полезность: 100%

## Ответ

Решение по обращению принимает полиция. Чтобы посмотреть ответ, перейдите в сервис Госуслуги → Мои заявки. Также по любым вопросам, связанным с обращением, вы можете позвонить по номеру 102. Это единый номер для связи с полицией в Казахстане.
