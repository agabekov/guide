# Где в мобильном приложении Kaspi.kz скачать заполненную декларацию по форме 270?

- Дата создания: 19.08.2025
- Дата обновления: 20.08.2025
- Полезность: 98%

## Ответ

Перейдите в сервис Госуслуги → Мои заявки → История заявок. Выберите нужную из списка и нажмите «Скачать декларацию по форме 270».
