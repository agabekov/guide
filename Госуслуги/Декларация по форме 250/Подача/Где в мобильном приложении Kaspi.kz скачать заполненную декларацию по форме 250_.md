# Где в мобильном приложении Kaspi.kz скачать заполненную декларацию по форме 250?

- Дата создания: 15.04.2024
- Дата обновления: 30.07.2025
- Полезность: 97%

## Ответ

Перейдите в сервис Госуслуги → Мои заявки → История заявок. Выберите нужную из списка и нажмите «Скачать декларацию по форме 250».
