# Я не вижу свои цифровые документы в мобильном приложении Kaspi.kz. Что делать?

- Дата создания: 31.03.2022
- Дата обновления: 31.03.2022
- Полезность: 83%

## Ответ

В мобильном приложении Kaspi.kz откройте сервис «Госуслуги» → Цифровые документы → Обновить список документов. Если после обновления вы не можете получить доступ к цифровым документам, обратитесь по номеру 1414. Информация по цифровым документам поступает в мобильное приложение Kaspi.kz из государственной базы данных.
