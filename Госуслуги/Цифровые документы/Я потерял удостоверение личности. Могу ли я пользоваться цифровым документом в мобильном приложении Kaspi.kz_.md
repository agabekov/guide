# Я потерял удостоверение личности. Могу ли я пользоваться цифровым документом в мобильном приложении Kaspi.kz?

- Дата создания: 31.03.2022
- Дата обновления: 31.03.2022
- Полезность: 95%

## Ответ

Нет. Вам нужно восстановить утерянный документ в ЦОНе, а затем обновить цифровые документы в мобильном приложении Kaspi.kz. Для этого откройте сервис «Госуслуги» → Цифровые документы → Обновить список документов. Информация по цифровым документам поступает в мобильное приложение Kaspi.kz из государственной базы данных. Если после обновления вы не видите в списке документов нужный, обратитесь по номеру 1414.
