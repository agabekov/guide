# Я получил новый документ. Как в мобильном приложении Kaspi.kz обновить цифровой документ?

- Дата создания: 31.03.2022
- Дата обновления: 31.03.2022
- Полезность: 93%

## Ответ

В мобильном приложении Kaspi.kz откройте сервис «Госуслуги» → Цифровые документы → Обновить список документов. Если после обновления у вас по-прежнему отображаются неверные документы, обратитесь по номеру 1414. Информация по цифровым документам поступает в мобильное приложение Kaspi.kz из государственной базы данных.
