# Как сделать отчисления для самозанятых за несколько месяцев в мобильном приложении Kaspi.kz?

- Дата создания: 21.12.2020
- Дата обновления: 28.12.2023
- Полезность: 96%

## Ответ

Чтобы оплатить обязательное социальное медицинское страхование или добровольные пенсионные взносы за несколько месяцев, нужно сделать несколько платежей — за каждый месяц отдельно. Для этого в <ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'> перейдите в сервис Госуслуги → Отчисления для самозанятых и выберите вид платежа.
