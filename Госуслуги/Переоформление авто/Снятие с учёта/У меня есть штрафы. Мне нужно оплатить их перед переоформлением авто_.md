# У меня есть штрафы. Мне нужно оплатить их перед переоформлением авто?

- Дата создания: 23.12.2020
- Дата обновления: 25.01.2021
- Полезность: 95%

## Ответ

Да. Если у вас есть задолженность по штрафам или налогу на транспорт, их нужно оплатить перед или в процессе переоформления авто. В мобильном приложении Kaspi.kz в сервисе «Госуслуги» → Штрафы вы можете сделать это тремя способами: по ИИН; по БИН; по госномеру авто.
