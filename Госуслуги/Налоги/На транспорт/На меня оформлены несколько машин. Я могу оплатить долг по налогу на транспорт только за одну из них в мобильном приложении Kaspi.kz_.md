# На меня оформлены несколько машин. Я могу оплатить долг по налогу на транспорт только за одну из них в мобильном приложении Kaspi.kz?

- Дата создания: 21.12.2020
- Дата обновления: 25.05.2022
- Полезность: 92%

## Ответ

Нет. В мобильном приложении Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'> отображается общий долг по определённому типу налога. Если у вас две машины, то сумма долга будет по налогу на обе сразу. Чтобы оплатить долг по одной из машин, нужно обратиться в управление государственных доходов по своему региону.
