# Долги по каким налогам можно оплатить в мобильном приложении Kaspi.kz?

- Дата создания: 21.12.2020
- Дата обновления: 25.05.2022
- Полезность: 87%

## Ответ

В мобильном приложении Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'> можно проверить и оплатить долг по всем налогам для физических лиц: Госуслуги → Налоги → Долги по налогам. Исключения: социальные отчисления, обязательные пенсионные взносы и социальное медицинское страхование. Чтобы узнать задолженность по этим платежам, нужно обратиться в ГЦВП. Долги по налогам для юридических лиц проверить на Kaspi.kz нельзя, но оплатить можно. Для этого перейдите в «Госуслуги» → «Налоги» → «Налог для юр. лиц по реквизитам» и заполните необходимые поля.
