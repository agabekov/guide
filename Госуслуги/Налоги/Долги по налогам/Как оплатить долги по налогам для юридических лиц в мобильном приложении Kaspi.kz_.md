# Как оплатить долги по налогам для юридических лиц в мобильном приложении Kaspi.kz?

- Дата создания: 21.12.2020
- Дата обновления: 25.05.2022
- Полезность: 81%

## Ответ

Перейдите в сервис «Госуслуги» → «Налоги» → «Налог для юр. лиц по реквизитам». Проверьте, что город указан верно, и выберите нужный тип платежа, управление государственных доходов. В поле «Период» выберите пункт «Долги прошлых лет». Введите БИН/ИИН организации, ФИО и ИИН налогоплательщика, укажите сумму, выберите способ оплаты и подтвердите платёж. После этого вы получите квитанцию об оплате и можете перейти во вкладку «История».
