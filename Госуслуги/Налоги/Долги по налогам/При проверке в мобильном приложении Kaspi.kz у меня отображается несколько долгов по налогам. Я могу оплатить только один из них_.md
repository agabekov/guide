# При проверке в мобильном приложении Kaspi.kz у меня отображается несколько долгов по налогам. Я могу оплатить только один из них?

- Дата создания: 21.12.2020
- Дата обновления: 25.05.2022
- Полезность: 87%

## Ответ

Да. Вы можете выбрать для оплаты только один из долгов. Для этого снимите галочки с тех задолженностей, которые вы не будете оплачивать сейчас.
