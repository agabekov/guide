# Заключение каких врачей нужно для справки №073/у?

- Дата создания: 22.07.2022
- Дата обновления: 19.02.2025
- Полезность: 96%

## Ответ

Справка №073у обязательно должна содержать: заключения терапевта, офтальмолога, оториноларинголога, психиатра; печати каждого из врачей и одну итоговую печать в заключении; перечисление всех категорий транспорта, которыми разрешено управлять; вашу фотографию 3×4 см. Пройти комиссию можно в любом медицинском учреждении. После этого врач внесет справку в государственную базу данных.
