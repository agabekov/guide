# Какие есть ограничения при переводах в мобильном приложении Kaspi.kz?

- Дата создания: 18.08.2016
- Дата обновления: 28.03.2025
- Полезность: 80%

## Ответ

Все переводы доступны 24/7. Ограничений по количеству переводов нет. При переводе с карты другого банка ограничение по сумме устанавливает банк, выпустивший карту. Также если карта другого банка не поддерживает технологию 3D Secure, то перевод совершить невозможно. Для подключения 3D Secure необходимо обратиться в банк, который выпустил карту.
