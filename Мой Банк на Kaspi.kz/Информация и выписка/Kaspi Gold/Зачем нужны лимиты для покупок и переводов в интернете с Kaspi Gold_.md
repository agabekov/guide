# Зачем нужны лимиты для покупок и переводов в интернете с Kaspi Gold?

- Дата создания: 18.08.2016
- Дата обновления: 04.11.2021
- Полезность: 95%

## Ответ

Лимит необходим для безопасности ваших покупок и переводов в интернете с Kaspi Gold. Выбранный лимит будет автоматически продлеваться каждый месяц. Исключение — лимит без ограничений. Его нужно в начале каждого месяца активировать заново самостоятельно.
