# Зачем при получении Автокредита на Kolesa.kz предоставлять данные контактных лиц?

- Дата создания: 18.08.2016
- Дата обновления: 10.10.2017
- Полезность: 98%

## Ответ

Контакты необходимы для подтверждения данных, которые вы указываете в анкете при получении кредита. Обязательств по кредиту контактные лица не несут.
