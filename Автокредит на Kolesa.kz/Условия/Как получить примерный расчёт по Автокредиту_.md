# Как получить примерный расчёт по Автокредиту?

- Дата создания: 18.08.2016
- Дата обновления: 19.10.2020
- Полезность: 96%

## Ответ

Вы можете получить предварительный расчёт по Автокредиту онлайн на сайте Kolesa.kz. Для этого выберите авто с отметкой «В кредит» и нажмите «Рассчитать кредит»: укажите размер первоначального взноса и выберите срок кредита. При подаче заявки онлайн решение по кредиту вы узнаете за 1 минуту.
