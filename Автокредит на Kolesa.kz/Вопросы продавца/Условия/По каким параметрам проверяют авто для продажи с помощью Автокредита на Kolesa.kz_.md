# По каким параметрам проверяют авто для продажи с помощью Автокредита на Kolesa.kz?

- Дата создания: 03.10.2019
- Дата обновления: 28.11.2024
- Полезность: 77%

## Ответ

Техническое состояние авто будет проверено по двум основным критериям: кузов, салон. Проверка автомобиля с пробегом нужна для того, чтобы Kaspi принял решение, какая сумма может быть выдана в кредит на выбранное авто, с учётом его рыночной стоимости и технического состояния.
