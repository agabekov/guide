# Где я могу посмотреть информацию по своему Социальному Счету в Kaspi.kz?

- Дата обновления: 27.09.2021

## Ответ

В мобильном приложении Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'> откройте сервис «Мой Банк» и выберите «Социальный Счет».
