# Может ли другой человек снять деньги с моего Социального Счета в Kaspi.kz?

- Дата создания: 27.09.2021
- Дата обновления: 27.09.2021
- Полезность: 81%

## Ответ

Нет. Никто, кроме вас, не сможет снять деньги. Снять наличные напрямую с Социального Счета можно только по вашему Kaspi QR. При снятии наличных в Kaspi Банкомате нужно пройти ещё несколько проверок, которые гарантируют безопасность ваших денег.
