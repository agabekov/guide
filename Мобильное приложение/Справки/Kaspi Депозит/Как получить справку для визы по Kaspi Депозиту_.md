# Как получить справку для визы по Kaspi Депозиту?

- Дата создания: 19.11.2024
- Дата обновления: 30.01.2025
- Полезность: 84%

## Ответ

Есть несколько способов получить справку для визы по Kaspi Депозиту: В <ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>. Перейдите в сервис Гид → Справки → Kaspi Депозит → О доступном остатке, укажите язык, на котором вам необходим документ, период и нажмите «Скачать». Чтобы распечатать справку, отправьте ее себе на почту с помощью любой программы, установленной на вашем смартфоне. Также справку можно получить в сервисе «Мой Банк». В любом отделении Kaspi.kz. Понадобятся документ, удостоверяющий личность, и мобильный телефон. Также вы можете самостоятельно распечатать справку в отделениях Kaspi.kz. Для этого отсканируйте QR-код на принтере, затем выберите справку «О доступном остатке» и следуйте дальнейшим шагам на экране. Справка выдается моментально и бесплатно. Она действительна на дату выдачи.
