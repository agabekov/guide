# При переводе в мобильном приложении Kaspi.kz с карты другого казахстанского банка на Kaspi Gold у меня запрашивают Secure Code. Как его получить?

- Дата создания: 18.08.2016
- Дата обновления: 12.06.2024
- Полезность: 91%

## Ответ

3D Secure Code — это дополнительный уровень безопасности при онлайн-платежах, который защищает вас от мошенничества. Одноразовый код приходит в сообщении от банка, выпустившего карту. С его помощью вы подтверждаете, что лично переводите деньги. Если сообщение с кодом не пришло, обратитесь в банк-отправитель.
