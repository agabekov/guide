# Снятие денег в Kaspi Банкомате по QR-коду это безопасно?

- Дата создания: 24.12.2018
- Дата обновления: 20.05.2019
- Полезность: 94%

## Ответ

Да. Никто, кроме вас, не сможет снять наличные по вашему QR-коду. QR-код используется только для авторизации. При снятии наличных в Kaspi Банкомате нужно пройти ещё несколько этапов верификации — это гарантирует безопасность операции.
