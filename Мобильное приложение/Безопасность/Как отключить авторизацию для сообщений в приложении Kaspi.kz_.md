# Как отключить авторизацию для сообщений в приложении Kaspi.kz?

- Дата создания: 14.05.2025
- Дата обновления: 14.05.2025
- Полезность: 96%

## Ответ

Вы можете отключить авторизацию для сообщений отдельно по каждому продукту в <ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>. Например, для Kaspi Gold или Kaspi Депозита. В сервисе «Сообщения» выберите нужный продукт — для этого в iOS проведите вдоль его названия справа налево, а в Android нажмите на него и удерживайте. Затем нажмите иконку-замок.
