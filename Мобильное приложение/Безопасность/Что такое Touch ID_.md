# Что такое Touch ID?

- Дата создания: 18.08.2016
- Дата обновления: 02.08.2023
- Полезность: 94%

## Ответ

Touch ID — способ идентификации пользователя по отпечатку пальца, разработанный компанией Apple. Аналогичная технология есть в большинстве современных телефонов с операционной системой Android. Сканер отпечатка пальца позволяет быстро разблокировать смартфон, авторизоваться в мобильных приложениях и надежно защищает личные данные.
