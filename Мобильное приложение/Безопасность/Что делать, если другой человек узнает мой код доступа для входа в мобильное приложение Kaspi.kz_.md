# Что делать, если другой человек узнает мой код доступа для входа в мобильное приложение Kaspi.kz?

- Дата создания: 27.03.2024
- Дата обновления: 28.03.2024
- Полезность: 98%

## Ответ

Вам необходимо сменить код доступа в настройках приложения Kaspi.kz. Для этого в строке меню выберите Сервисы → Вход/Регистрация. Затем нажмите Настройки → Изменить код доступа и следуйте дальнейшим шагам. Для безопасности кодом доступа можно пользоваться только на том устройстве, где вы его настроили. Другой человек не сможет войти в ваше мобильное приложение Kaspi.kz со своего устройства, даже если будет знать код доступа.
