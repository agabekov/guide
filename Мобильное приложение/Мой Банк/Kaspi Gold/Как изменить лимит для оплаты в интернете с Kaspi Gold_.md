# Как изменить лимит для оплаты в интернете с Kaspi Gold?

- Дата создания: 18.08.2016
- Дата обновления: 05.10.2020
- Полезность: 93%

## Ответ

Вы можете изменить лимит оплаты по карте Kaspi Gold для покупок в интернете на Kaspi.kz: Мой Банк → Kaspi Gold → Действия → Оплата в интернете. Новый лимит можно использовать сразу после изменения.
