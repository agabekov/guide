# Кто может стать доверенным контактом в приложении Kaspi.kz?

- Дата создания: 30.06.2025
- Дата обновления: 01.07.2025
- Полезность: 88%

## Ответ

Доверенным контактом для своих родителей может стать только ребенок старше 21 года. Родителям нужно в приложении Kaspi.kz открыть Настройки → Добавить доверенный контакт и указать номер телефона ребенка.
