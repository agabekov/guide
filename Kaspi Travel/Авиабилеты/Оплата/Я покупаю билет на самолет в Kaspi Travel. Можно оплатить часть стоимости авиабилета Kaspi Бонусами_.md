# Я покупаю билет на самолет в Kaspi Travel. Можно оплатить часть стоимости авиабилета Kaspi Бонусами?

- Дата создания: 30.11.2020
- Дата обновления: 17.07.2024
- Полезность: 91%

## Ответ

Да. Вы можете использовать для покупки авиабилета накопленные Kaspi Бонусы. Для этого при оплате отметьте «Потратить бонусы». Стоимость покупки уменьшится на сумму бонусов. Исключение: при оплате со Счета Kaspi Pay трата бонусов недоступна.
