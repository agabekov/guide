# Я купил авиабилет через Kaspi Travel. Могу добавить в этот билет нового пассажира?

- Дата создания: 30.11.2020
- Дата обновления: 09.06.2021
- Полезность: 66%

## Ответ

К сожалению, добавить нового пассажира в уже купленный билет невозможно. Вы можете оформить отдельный билет для нового пассажира.
