# Я собираюсь лететь с младенцем. Для него нужно покупать отдельный авиабилет на Kaspi Travel?

- Дата создания: 30.09.2022
- Дата обновления: 05.10.2022
- Полезность: 88%

## Ответ

Да. Вам нужно оформить билет для младенца. Есть два варианта сделать это: во время покупки авиабилета; добавить младенца в уже купленный билет. Билет может быть платным или бесплатным, в зависимости от компании-перевозчика. Младенцем считается ребёнок до 2 лет. Он летает с родителями без предоставления отдельного места.
