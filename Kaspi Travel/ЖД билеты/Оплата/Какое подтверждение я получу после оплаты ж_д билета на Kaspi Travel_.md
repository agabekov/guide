# Какое подтверждение я получу после оплаты ж/д билета на Kaspi Travel?

- Дата создания: 10.06.2021
- Дата обновления: 10.07.2024
- Полезность: 76%

## Ответ

Подтверждением оплаты служит фискальный чек. Чтобы посмотреть его, отсканируйте QR-код на вашем электронном билете. Для этого вы можете переслать билет с помощью любого мессенджера, открыть его на другом устройстве или распечатать, а затем отсканировать с помощью камеры смартфона.
