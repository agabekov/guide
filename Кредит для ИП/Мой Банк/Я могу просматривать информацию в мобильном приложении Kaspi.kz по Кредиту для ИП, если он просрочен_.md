# Я могу просматривать информацию в мобильном приложении Kaspi.kz по Кредиту для ИП, если он просрочен?

- Дата создания: 24.08.2020
- Дата обновления: 03.11.2025
- Полезность: 100%

## Ответ

Да. Вы сможете просматривать информацию по Кредиту для ИП в разделе «Мой Банк» в <ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>. Сумму долга, расписание и сумму платежей можно посмотреть во вкладке «Ежемесячный платеж». Во вкладке Кредиты → Информация о кредите вы можете уточнить следующие данные: • сумма, дата получения и срок кредита, • размер переплаты, • минимальная сумма для частичного досрочного погашения, • сумма задолженности.
