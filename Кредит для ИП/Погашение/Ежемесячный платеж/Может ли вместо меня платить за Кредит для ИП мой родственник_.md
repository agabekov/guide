# Может ли вместо меня платить за Кредит для ИП мой родственник?

- Дата создания: 27.08.2020
- Дата обновления: 14.12.2020
- Полезность: 100%

## Ответ

Да. Родственник может оплатить ежемесячный платеж по вашему кредиту. Для оплаты достаточно знать номер вашего ИИН. Оплату можно провести в любом Kaspi Терминале или Kaspi Банкомате без комиссий и без очередей.
