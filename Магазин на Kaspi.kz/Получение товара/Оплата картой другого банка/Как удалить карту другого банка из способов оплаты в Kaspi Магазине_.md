# Как удалить карту другого банка из способов оплаты в Kaspi Магазине?

- Дата создания: 12.08.2025
- Дата обновления: 13.08.2025
- Полезность: 100%

## Ответ

При выборе способа оплаты нажмите на три точки рядом с картой другого банка, затем — «Удалить». Обратите внимание: обновление приложения Kaspi.kz будет проходить постепенно.
