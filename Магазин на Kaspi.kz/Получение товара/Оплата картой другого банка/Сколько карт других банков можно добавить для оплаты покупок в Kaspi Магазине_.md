# Сколько карт других банков можно добавить для оплаты покупок в Kaspi Магазине?

- Дата создания: 12.08.2025
- Дата обновления: 13.08.2025
- Полезность: 93%

## Ответ

Вы можете добавить только одну карту другого банка. Чтобы использовать новую, удалите предыдущую карту из способов оплаты в Kaspi Магазине. Обратите внимание: обновление приложения Kaspi.kz будет проходить постепенно.
