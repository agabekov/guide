# Мне не перезвонили из магазина по моему заказу. Что делать?

- Дата создания: 18.08.2016
- Дата обновления: 15.05.2024
- Полезность: 63%

## Ответ

Есть несколько способов узнать, подтвержден ли ваш заказ: в мобильном приложении Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>: Магазин → Мои заказы→ Детали заказа; по номеру телефона магазина, указанному в сообщении о заказе.
