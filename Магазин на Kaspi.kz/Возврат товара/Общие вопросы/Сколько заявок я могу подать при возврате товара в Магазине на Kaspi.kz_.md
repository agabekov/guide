# Сколько заявок я могу подать при возврате товара в Магазине на Kaspi.kz?

- Дата создания: 16.05.2023
- Дата обновления: 15.05.2024
- Полезность: 56.99999999999999%

## Ответ

В мобильном приложении Kaspi.kz вы можете отправить любое количество заявок на возврат. В каждой из них можно указать только один вид товара. Для этого откройте сервис Магазин на Kaspi.kz → Мои заказы. Выберите нужную покупку, нажмите «Оформить возврат» и заполните заявку, следуя инструкциям на экране. Вы сможете в любое время проверить статус заявки, этапы возврата и точные сроки в сервисе Магазин на Kaspi.kz → Мои заказы → Заявки на возврат → Детали заявки. Если в возврате будет отказано, вам придет сообщение в мобильном приложении Kaspi.kz с указанием причины.
