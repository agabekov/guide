# Как добавить машину из госбазы, чтобы подобрать к ней запчасти в Kaspi Магазине?

- Дата создания: 14.10.2025
- Дата обновления: 14.10.2025
- Полезность: 100%

## Ответ

В приложении Kaspi.kz перейдите в Магазин → Автотовары → Автозапчасти. Нажмите на поле «VIN-код авто или артикул запчасти», чтобы открыть окно подбора деталей. В нем выберите «Добавить мое авто из Госбазы» и следуйте дальнейшим шагам на экране. После этого в категории «Автозапчасти» отобразятся ваши машины. Выберите ту, для которой необходимо подобрать запчасти — в списке товаров останутся только подходящие для неё детали.
