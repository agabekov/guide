# Я добавил получателя для заказа из Kaspi Магазина с доставкой. Как его поменять?

- Дата создания: 26.05.2025
- Дата обновления: 26.05.2025
- Полезность: 81%

## Ответ

Чтобы поменять получателя, в сервисе Магазин → Мои заказы выберите нужный и перейдите в «Детали заказа». Нажмите «Изменить» рядом с именем человека, которого вы добавили ранее. Укажите номер телефона нового получателя — он должен быть зарегистрирован в Kaspi.kz.
