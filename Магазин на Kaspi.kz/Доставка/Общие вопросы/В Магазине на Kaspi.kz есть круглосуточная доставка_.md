# В Магазине на Kaspi.kz есть круглосуточная доставка?

- Дата создания: 14.02.2024
- Дата обновления: 15.02.2024
- Полезность: 78%

## Ответ

В Магазине на Kaspi.kz можно заказать круглосуточную доставку только из аптек, которые работают 24/7. Для этого в Магазине на Kaspi.kz откройте нужный товар, затем в фильтре отметьте «Express, за 3 часа». Выберите аптеку и добавьте товар в корзину. Далее вы сможете указать промежуток времени, в который хотите получить заказ.
