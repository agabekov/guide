# Другой человек может получить мой заказ с доставкой из Kaspi Магазина?

- Дата создания: 26.05.2025
- Дата обновления: 07.12.2025
- Полезность: 100%

## Ответ

Да. Другой человек может получить ваш заказ из Kaspi Магазина, если он клиент Kaspi.kz и старше 16 лет. Для этого назначьте его доверенным получателем. Это можно сделать в любой момент после оформления заказа. Перейдите в Магазин → Мои заказы. Выберите нужный и нажмите Детали заказа → Добавить получателя. Укажите его номер телефона — он должен быть зарегистрирован в Kaspi.kz.
