# Как добавить доверенного получателя для заказа в Kaspi Postomat?

- Дата создания: 16.07.2024
- Дата обновления: 26.05.2025
- Полезность: 90%

## Ответ

Вы можете сделать это в любое время после оформления заказа. Перейдите в Магазин → Мои заказы. Выберите нужный и нажмите Детали заказа → Добавить получателя. Укажите его номер телефона — он должен быть зарегистрирован в Kaspi.kz. Получателем может стать клиент Kaspi.kz старше 16 лет.
