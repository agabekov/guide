# Сколько получателей можно добавить для заказа в Kaspi Postomat?

- Дата создания: 16.07.2024
- Дата обновления: 26.05.2025
- Полезность: 95%

## Ответ

Вы можете добавить только одного доверенного получателя. Если в ячейке Kaspi Postomat размещено несколько заказов, доверенному получателю необходимо забрать их все.
