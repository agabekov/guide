# Другой человек может забрать мой заказ из Kaspi Postоmat?

- Дата создания: 09.11.2021
- Дата обновления: 26.05.2025
- Полезность: 94%

## Ответ

Да. Другой человек может забрать ваш заказ в Kaspi Postomat, если он клиент Kaspi.kz и старше 16 лет. Для этого назначьте его доверенным получателем. Это можно сделать в любой момент после оформления заказа. Перейдите в Магазин → Мои заказы. Выберите нужный и нажмите Детали заказа → Добавить получателя. Укажите его номер телефона — он должен быть зарегистрирован в Kaspi.kz. Получателю придет сообщение в приложении Kaspi.kz, где будут указаны товары, которые нужно забрать, и срок их хранения. Чтобы открыть ячейку, нужно с помощью приложения Kaspi.kz отсканировать QR-код на Kaspi Postomat.
