# Срок действия моего Накопительного Kaspi Депозита истекает. Как его продлить?

- Дата создания: 24.02.2025
- Дата обновления: 18.04.2025
- Полезность: 99%

## Ответ

Накопительный Kaspi Депозит автоматически продлится на тот же срок, и вам не нужно ничего для этого делать. Ставка вознаграждения будет такой, которая действует на момент пролонгации. Ограничений по количеству продлений нет. Дату будущего продления, а также другую информацию по вкладу вы можете в любой момент посмотреть в сервисе «Мой Банк»: выберите нужный депозит, затем Инфо → Условия Накопительного Депозита.
