# Можно открыть депозит в Kaspi.kz по внутреннему паспорту другой страны?

- Дата создания: 24.02.2025
- Дата обновления: 26.02.2025
- Полезность: 89%

## Ответ

Нет. Граждане другой страны могут открыть депозит только по заграничному паспорту. Внутренний паспорт другой страны не является документом, удостоверяющим личность, в Республике Казахстан.
