# Может ли другой человек снять деньги в Kaspi Банкомате с моего счёта по моему Kaspi QR?

- Дата создания: 06.08.2019
- Дата обновления: 30.09.2025
- Полезность: 93%

## Ответ

Нет. Никто, кроме вас, не сможет снять наличные по вашему Kaspi QR. Kaspi QR используется только для авторизации. При снятии наличных в Kaspi Банкомате нужно пройти ещё несколько проверок, которые гарантируют безопасность ваших денег. Например, подтверждение с помощью фотоверификации.
