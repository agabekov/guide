# Я сделал покупку через сервис Alipay+ с Kaspi Gold. Где я могу посмотреть информацию по ней?

- Дата создания: 05.03.2025
- Дата обновления: 05.03.2025
- Полезность: 95%

## Ответ

Это можно сделать двумя способами: • Мой банк → Kaspi Gold → Выписка. Укажите период и выберите необходимую покупку; • Главная → Сообщения → Kaspi Gold.
