# Какие ограничения есть при пополнении Kaspi Gold через сервис Visa по номеру телефона с карты другого банка?

- Дата создания: 22.11.2024
- Дата обновления: 14.01.2025
- Полезность: 99%

## Ответ

Kaspi Gold можно пополнить с карты другого банка в любой валюте. При этом сумма на Kaspi Gold будет переведена в тенге. Комиссии, сроки, ограничения по сумме может установить банк-отправитель. Kaspi.kz не взимает никаких комиссий за пополнение Kaspi Gold.
