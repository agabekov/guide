# При покупке в интернете с Kaspi Gold у меня не запрашивали 3D Secure Code. Почему?

- Дата создания: 18.11.2021
- Дата обновления: 18.11.2021
- Полезность: 82%

## Ответ

На некоторых сайтах 3D Secure автоматически проверяет операцию и подтверждает её без ввода кода. Если ресурс не поддерживает эту технологию, вводить 3D Secure Code не требуется, поэтому сообщение с ним не приходит.
