# Я нахожусь за границей. Хочу сделать покупку в интернете с Kaspi Gold. Мне придёт сообщение с 3D Secure Code?

- Дата создания: 18.11.2021
- Дата обновления: 18.11.2021
- Полезность: 76%

## Ответ

Да. Вам придёт сообщение с 3D Secure Code, если ваш телефон подключен к интернет-соединению или роумингу. Для безопасности ваших денег код придёт вам в сообщении на номер, который вы использовали при оформлении Kaspi Gold. Его нужно ввести для подтверждения, что операцию совершаете именно вы. На некоторых сайтах 3D Secure автоматически проверяет операцию и подтверждает её без ввода кода. Если ресурс не поддерживает эту технологию, вводить 3D Secure Code не требуется, поэтому сообщение с ним не приходит.
