# Я неправильно ввёл 3D Secure Code при покупке в интернете с Kaspi Gold. Что делать?

- Дата создания: 18.11.2021
- Дата обновления: 18.11.2021
- Полезность: 83%

## Ответ

Вам нужно повторно ввести 3D Secure Code. Если код будет введён неправильно три раза, то, для безопасности ваших денег, операция будет отменена. Чтобы совершить оплату, вам понадобится повторно ввести данные Kaspi Gold.
