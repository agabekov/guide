# Что такое 3D Secure?

- Дата создания: 18.11.2021
- Дата обновления: 18.11.2021
- Полезность: 90%

## Ответ

3D Secure — это технология, которая обеспечивает дополнительную безопасность покупок и переводов в интернете с Kaspi Gold. Когда вы платите картой в интернете, вам приходит сообщение с одноразовым шестизначным кодом — это 3D Secure Code. Его нужно ввести для подтверждения, что операцию совершаете именно вы. Сообщение с кодом вы получите на свой номер, который указали при оформлении Kaspi Gold. На некоторых сайтах 3D Secure автоматически проверяет операцию и подтверждает её без ввода кода. Если ресурс не поддерживает эту технологию, вводить 3D Secure Code не требуется, поэтому сообщение с ним не приходит. 3D Secure для владельцев Kaspi Gold подключается всегда бесплатно.
