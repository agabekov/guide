# Хочу сделать покупку с Kaspi Gold в интернете. Как узнать, поддерживает ли сайт 3D Secure?

- Дата создания: 18.11.2021
- Дата обновления: 14.03.2024
- Полезность: 89%

## Ответ

На сайтах и в приложениях, которые поддерживают 3D Secure, размещён логотип Verified by Visa или Mastercard SecureCode. На некоторых сайтах 3D Secure автоматически проверяет операцию и подтверждает её без ввода кода. Если ресурс не поддерживает эту технологию, вводить 3D Secure Code не требуется, поэтому сообщение с ним не приходит.
