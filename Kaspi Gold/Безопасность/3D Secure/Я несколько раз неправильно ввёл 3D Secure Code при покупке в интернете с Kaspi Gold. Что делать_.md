# Я несколько раз неправильно ввёл 3D Secure Code при покупке в интернете с Kaspi Gold. Что делать?

- Дата создания: 18.11.2021
- Дата обновления: 18.11.2021
- Полезность: 95%

## Ответ

Если вы ввели код неправильно три раза, то, для безопасности ваших денег, покупка или перевод будут отменены. Чтобы совершить оплату, вам понадобится повторно ввести данные Kaspi Gold.
