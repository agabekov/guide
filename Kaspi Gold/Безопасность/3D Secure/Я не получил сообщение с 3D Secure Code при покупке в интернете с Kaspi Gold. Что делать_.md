# Я не получил сообщение с 3D Secure Code при покупке в интернете с Kaspi Gold. Что делать?

- Дата создания: 18.11.2021
- Дата обновления: 18.11.2021
- Полезность: 63%

## Ответ

На некоторых сайтах 3D Secure автоматически проверяет операцию и подтверждает её без ввода кода. Если ресурс не поддерживает эту технологию, вводить 3D Secure Code не требуется, поэтому сообщение с ним не приходит. Если открылась специальная страница 3D Secure, и сообщение с кодом не пришло, нажмите «Отправить код повторно». После этого вам придёт сообщение с кодом.
