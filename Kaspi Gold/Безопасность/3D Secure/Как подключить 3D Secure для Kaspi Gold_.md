# Как подключить 3D Secure для Kaspi Gold?

- Дата создания: 18.11.2021
- Дата обновления: 18.11.2021
- Полезность: 84%

## Ответ

3D Secure автоматически подключается к Kaspi Gold. Вам ничего не нужно делать для этого. 3D Secure Code придёт вам в сообщении, когда вы будете совершать покупку или перевод в интернете с Kaspi Gold. На некоторых сайтах 3D Secure автоматически проверяет операцию и подтверждает её без ввода кода. Если ресурс не поддерживает эту технологию, вводить 3D Secure Code не требуется, поэтому сообщение с ним не приходит.
