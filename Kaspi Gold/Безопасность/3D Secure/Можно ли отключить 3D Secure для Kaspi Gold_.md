# Можно ли отключить 3D Secure для Kaspi Gold?

- Дата создания: 18.11.2021
- Дата обновления: 18.11.2021
- Полезность: 69%

## Ответ

Нет. Отключить 3D Secure невозможно. На некоторых сайтах 3D Secure автоматически проверяет операцию и подтверждает её без ввода кода. Если ресурс не поддерживает эту технологию, вводить 3D Secure Code не требуется, поэтому сообщение с ним не приходит.
