# Kaspi Gold поддерживает 3D Secure?

- Дата создания: 18.11.2021
- Дата обновления: 18.11.2021
- Полезность: 79%

## Ответ

Да. 3D Secure автоматически подключен к Kaspi Gold. 3D Secure Code придёт в сообщении на ваш номер, когда вы будете совершать покупку или перевод с карты в интернете. Его нужно ввести для подтверждения, что операцию совершаете именно вы. Сообщение с кодом вы получите на свой номер, который указали при оформлении Kaspi Gold. На некоторых сайтах 3D Secure автоматически проверяет операцию и подтверждает её без ввода кода. Если ресурс не поддерживает эту технологию, вводить 3D Secure Code не требуется, поэтому сообщение с ним не приходит.
