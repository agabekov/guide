# Сколько времени действует 3D Secure Code для покупки в интернете с Kaspi Gold?

- Дата создания: 18.11.2021
- Дата обновления: 18.11.2021
- Полезность: 93%

## Ответ

3D Secure Code действует 10 минут. В течение этого времени нужно его ввести, чтобы подтвердить покупку или перевод в интернете. Если время истекло, нужно будет совершить оплату заново.
