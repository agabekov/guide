# Какие реквизиты нужно знать, чтобы перевести деньги c Kaspi Gold на карту другого банка Казахстана?

- Дата создания: 22.06.2017
- Дата обновления: 13.07.2018
- Полезность: 87%

## Ответ

Достаточно знать только номер карты получателя. После перевода вы можете поставить отметку «Сохранить в Частые», чтобы не вводить номер карты в следующий раз.
