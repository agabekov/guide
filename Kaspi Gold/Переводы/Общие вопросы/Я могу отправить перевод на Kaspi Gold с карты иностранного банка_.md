# Я могу отправить перевод на Kaspi Gold с карты иностранного банка?

- Дата создания: 22.06.2017
- Дата обновления: 08.01.2025
- Полезность: 94%

## Ответ

Да. Вы можете отправить перевод на Kaspi Gold через сайт или приложение зарубежного банка, если он поддерживает такие операции. Есть несколько способов: по номеру Kaspi Gold. Реквизиты для перевода вы можете посмотреть в мобильном приложении Kaspi.kz: Мой Банк → Kaspi Gold → Инфо → Реквизиты карты и счета. по номеру телефона. Получателю необходимо подключить сервис «Пополнения с Visa по номеру телефона» в мобильном приложении Kaspi.kz: Мой Банк → Kaspi Gold → Инфо → Пополнения с Visa. Обратите внимание, условия: комиссии, сроки, ограничения по сумме — устанавливает банк-отправитель. Kaspi.kz не взимает никаких комиссий за пополнение Kaspi Gold.
