# Я могу отправить получателю сообщение во время перевода денег в мобильном приложении Kaspi.kz?

- Дата создания: 22.06.2017
- Дата обновления: 26.03.2022
- Полезность: 93%

## Ответ

Да. Вы можете написать получателю сообщение при переводах на Kaspi Gold других клиентов. Максимальная длина сообщения — 80 символов. Сообщение можно добавить по желанию, это поле не обязательно для заполнения. При других переводах эта функция пока недоступна.
