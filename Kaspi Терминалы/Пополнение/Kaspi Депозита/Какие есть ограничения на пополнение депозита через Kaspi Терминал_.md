# Какие есть ограничения на пополнение депозита через Kaspi Терминал?

- Дата создания: 18.08.2016
- Дата обновления: 05.04.2022
- Полезность: 88%

## Ответ

Kaspi Терминалы принимают только купюры тенге. Пополнение монетами или другой валютой невозможно. При пополнении по ИИН действует ограничение по сумме — 500 000 тенге в день. Если вы авторизуетесь в терминале с помощью любой карты Kaspi, то ограничений по сумме не будет.
