# Что нужно для оплаты кредита через Kaspi Терминал?

- Дата создания: 18.08.2016
- Дата обновления: 07.04.2022
- Полезность: 87%

## Ответ

Для оплаты кредита через Kaspi Терминал вам нужно авторизоваться. Это можно сделать любым из трёх способов: через мобильное приложение Kaspi.kz. Откройте сервис Kaspi QR и отсканируйте QR-код на экране Kaspi Терминала. вставить любую карту Kaspi. ввести ИИН. Этим способом можно также оплатить кредит другого человека.
