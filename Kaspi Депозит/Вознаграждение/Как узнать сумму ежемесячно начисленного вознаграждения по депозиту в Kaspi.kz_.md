# Как узнать сумму ежемесячно начисленного вознаграждения по депозиту в Kaspi.kz?

- Дата создания: 18.08.2016
- Дата обновления: 26.02.2025
- Полезность: 93%

## Ответ

Бесплатный сервис «Сообщения» будет уведомлять вас в начале месяца о начисленном вознаграждении и доступной сумме на депозитах в Kaspi.kz. Вы можете посмотреть детальный расчёт процентов по вашим депозитам за каждый месяц в <ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>. Для этого в сервисе «Мой Банк» выберите нужный депозит, откройте вкладку «Выписка» и нажмите строку с информацией о начислении вознаграждения.
