# Kaspi.kz повысил ставку вознаграждения по депозитам в тенге. Что будет с моим Kaspi Депозитом?

- Дата создания: 07.02.2022
- Дата обновления: 10.04.2025
- Полезность: 99%

## Ответ

Процентная ставка вашего Kaspi Депозита изменится автоматически, от вас никаких действий не требуется. Посмотреть действующую ставку по своему депозиту можно в мобильном приложении Kaspi.kz в сервисе «Мой Банк». Для этого выберите нужный депозит и перейдите во вкладку Инфо → Условия Kaspi Депозита.
