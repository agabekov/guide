# Срок действия моего Kaspi Депозита истекает. Как его продлить?

- Дата создания: 18.08.2016
- Дата обновления: 24.02.2025
- Полезность: 99%

## Ответ

Kaspi Депозит автоматически продлится еще на год, и вам не нужно ничего для этого делать. Ставка вознаграждения будет такой, которая действует на момент пролонгации. Ограничений по количеству продлений нет. Дату будущего продления, а также другую информацию по вкладу вы можете в любой момент посмотреть в сервисе «Мой Банк»: выберите нужный депозит, затем Инфо → Условия Kaspi Депозита.
