# Как пополнить Kaspi Депозит через Kaspi Терминал?

- Дата создания: 18.08.2016
- Дата обновления: 11.05.2023
- Полезность: 94%

## Ответ

Для пополнения депозита через Kaspi Терминал нужно авторизоваться. Это можно сделать любым из трёх способов: через мобильное приложение Kaspi.kz<ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'><ссылка: ссылка_приложения/iOS: 'https://appsto.re/kz/zU2ohb.i'; android: 'https://play.google.com/store/apps/details?id=kz.kaspi.mobile'>. Откройте сервис Kaspi QR и отсканируйте QR-код на экране Kaspi Терминала. с помощью любой карты Kaspi.kz. Вставьте карту в картридер терминала. ввести ИИН. Введите свой ИИН или человека, которому пополняете Kaspi Депозит. Через Kaspi Терминал можно пополнить только тенговый депозит.
